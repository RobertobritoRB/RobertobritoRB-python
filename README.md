Relatório – Sistema de Gerenciamento de Livros
Introdução

O objetivo desta atividade foi desenvolver um sistema simples de gerenciamento de livros utilizando a linguagem Python. 

O sistema permite cadastrar livros, listar os livros cadastrados, realizar busca por título e gerar um gráfico com a
quantidade de livros por gênero utilizando a biblioteca Matplotlib.

Lógica utilizada

Inicialmente foi criada uma classe chamada Livro, responsável por representar cada livro do sistema. Essa classe possui atributos como título, autor,
gênero e quantidade disponível.

Em seguida foi criada uma lista chamada biblioteca, responsável por armazenar todos os objetos da classe Livro cadastrados no sistema.

Foram implementadas funções para realizar as principais operações do sistema:

Cadastrar livro: solicita as informações ao usuário e cria um novo objeto da classe Livro, adicionando-o à lista.

Listar livros: percorre a lista de livros e exibe suas informações.

Buscar livro: permite localizar um livro pelo título informado pelo usuário.

Gerar gráfico: utiliza a biblioteca Matplotlib para gerar um gráfico de barras mostrando a quantidade de livros por gênero.

Por fim, foi criado um menu interativo, utilizando estruturas de repetição e condicionais, permitindo que o usuário 
escolha as funcionalidades do sistema até decidir encerrar o programa.

Conclusão

Com o desenvolvimento deste sistema foi possível aplicar conceitos fundamentais da programação em Python, como orientação a objetos, listas, funções, 
estruturas condicionais e de repetição, além da utilização da biblioteca Matplotlib para visualização de dados em forma de gráfico.
